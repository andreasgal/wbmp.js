This tests a basic WBMP image that has been corrupted with a binary editor.
The WBMP decoder currently only deals with a well-defined type 0 WBMP.

Good image - width set to 1

Bad image - type field only

Bad image - fix header bit 1 is set
Bad image - fix header bit 2 is set
Bad image - fix header bit 3 is set
Bad image - fix header bit 4 is set

Bad image - type and fix header fields only
Bad image - huge width
Bad image - type, fix header and width only
Bad image - huge height
